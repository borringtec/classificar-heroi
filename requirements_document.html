1. Objetivos

O Classificador de Nível de Herói é um sistema que visa classificar heróis de acordo com a quantidade de experiência (XP) que possuem, apresentando um nível correspondente.

1.1. Objetivos do Documento

Este documento tem como objetivo descrever e especificar os requisitos do sistema, servindo como guia para os desenvolvedores e interessados.

2. Escopo Geral do Produto

O sistema classifica heróis com base em sua experiência, exibindo mensagens correspondentes ao nível alcançado.

3. Convenções, Termos e Abreviações

  • XP – Experiência do herói.
  • Nível – Classificação do herói com base na XP.

4. Requisitos

4.1. Requisitos Funcionais

RF001 Classificar o herói com base na XP fornecida.
Entrada Nome do herói e quantidade de XP.
Saída Nível do herói.
Pré-condição Informações de nome e XP fornecidas.
Pós-condição Classificação do herói realizada com sucesso.

4.2 Requisitos Não Funcionais

RNF001 O sistema deve ser responsivo e funcionar em dispositivos móveis.

4.3 Requisitos de Domínio

RD001

Se XP for menor do que 1.000 = Ferro

Se XP for entre 1.001 e 2.000 = Bronze

Se XP for entre 2.001 e 5.000 = Prata

Se XP for entre 5.001 e 7.000 = Ouro

Se XP for entre 7.001 e 8.000 = Platina

Se XP for entre 8.001 e 9.000 = Ascendente

Se XP for entre 9.001 e 10.000= Imortal

Se XP for maior ou igual a 10.001 = Radiante
